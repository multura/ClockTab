# ClockTab
Веб-приложение с крупными цифровыми часами на фоне анимированного слайд-шоу. Часы отображают текущее время и дату, а на заднем плане плавно сменяются изображения с соответствующими описаниями.

## Особенности

    Крупные часы - отображение текущего времени в реальном времени

    Слайд-шоу - три фоновых изображения с плавными переходами

    Описания - для каждого слайда есть текстовое описание

    Собственный шрифт - использование шрифта SF Pro с разными начертаниями (не гарантированна работа с IE)

    Адаптивный дизайн - корректное отображение на всех устройствах

    Совместимость - поддержка веб-движков с 2012 года

***В первую очередь был создан для работы с планшетом Surface RT, но также может работать и с любыми девайсами.***
